# Getting started
<hr>
1. Connect your device to teh computer with USB cable 
2. Download the Galaxy_Ninja folder
3. Open it in Unity
4. In Unity -> press File -> Build Settings
5. Switch to iOS or Android platform in Build Settings
6. Build and Run
7. Enjoy :)
